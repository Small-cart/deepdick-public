## 部署

建议使用Docker Compose

```bash
git clone https://github.com/tgx-coding/deepdick-public
cd deepdick-public
docker compose up -d
```
使用前请在`docker-compose.yml`中配置`username` `password` `API_KEY` `parents_name`环境变量

或者直接克隆本项目运行

```
git clone https://github.com/tgx-coding/deepdick-public
cd deepdick-public
pip install -r requirements.txt
python main.py
```

请注意，直接部署时请手动替换`main.py`中的环境变量
